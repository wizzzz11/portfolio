How it works

  1. Choose Image or Video tab and upload a file.
  2. Preview will show the media and sample frames (for video).
  3. Click Analyze — the front end sends the file to your detection API.
  4. Backend should respond with JSON: {verdict: 'real'|'fake', confidence:0.92, details: '...'}
Integration tips
  • POST to /api/deepfake/analyze with a FormData field file.
  • Return JSON with verdict (string) and confidence (0-1 float).
